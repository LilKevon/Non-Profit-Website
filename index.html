About Us!

Output("Success!") is a youth-led, non-profit organization devoted to ensuring quality computer science lessons to all students. Through FREE lessons and 1 on 1 private sessions, we ensure that future programmers thrive and add new languages into their arsenal. We hope to see new students eager to dive into the realms of Computer Programming.

Students may struggle to find the right support academically, especially when it comes to learning a new programming language. There may be some who are curious about programming, but fail to find helpful resources online. Our mission is to empower these students, and guide them as they start their journey in the realm of computer programming. We also strive to make this support easy to access, and ensure that our students are comfortable when learning.

Why Choose Us?

We're completely free!

Programming lessons should be accessible for everyone. Here at Output("Success!"), we ensure that your child's education in programming is free, lifting the financial burden off your shoulders!

Friendly, Qualified Tutors

Our tutors remember what it was like to start coding, and know the obstacles and challenges students face when starting out. Our application process ensures that your child's tutor is top of the notch, and friendly to boot.

Class Lessons & 1 on 1

Knowing that some students that have different learning strategies, we host both class lessons, and 1 on 1 tutoring sessions. In class lessons, students are in groups of 3-5, and learn together. In 1 on 1 sessions, students will have their own tutor, and are able to learn without the pressures of being in a class.

100% Safe and Virtual Lessons

All of our lesssons are hosted on Discord, a communication platform which most children are completely comfortable with. This ensures that all lessons are completely safe, especially during these tough times.

Programs We Offer

Here at Output("Success!"), we offer several courses for Java, one of the most popular languages in the world. In time, other programming languages will be added into our programs. Best of all, all these lessons are free!

Java I: Introduction to Basics

A beginner friendly program, Java I covers Java's general topics. Anyone can take this course, and will teach the basics, including:

-Installing an IDE
-Output
-Variable and Data Types
-Control Structures(For, While)
-Decision Structures(If, Switch)
-Nested Structures
-Arrays
-String Handling
-Basic Data Structures




All students welcome to try this program out!


PRICE: FREE

DURATION: 90 MIN

Java II: Advanced Java Programming

This is a course for the more experienced programmers. This course covers advanced topics in Java, and is perfect for strengthening your Java knowledge. These topics include:

-Exception Handling
-The Vast Java API Library
-Data Structures
-Threads
-Annotations
-Enumerations
-Auto Boxing
-Basic Algorithms
-Generics
-Lambda Expressions

Students are encouraged to try Java I before applying for this program.

PRICE: FREE

DURATION: 90 MIN

Java III: Class Design

This program covers the fundamentals of Class Design in Java. With Class Design, students can learn the inner workings of OOP programming, and learn to create objects of their own. This course includes:

-Class Creation
-Constructors
-Methods
-Modifiers & Identifiers
-Method Overriding & Overloading
-Inheritance
-Encapsulation
-Polymorphism
-Interfaces


Students are encouraged to try Java I and II before applying for this program.

PRICE: FREE

DURATION: 90 MIN

Java IV: GUI Design

Who doesn't like creating applications and games? Java IV dives into Swing, Java's main GUI interface. During this course, students will learn to create powerful GUIs, and play test these GUI's themselves. These program covers:

-What is Swing/GUI?
-Swing Components
-AWT Events
-Creating Applications
-Creating Games





Students are encouraged to try Java I and Java II before applying for this program.


PRICE: FREE

DURATION: 90 MIN
